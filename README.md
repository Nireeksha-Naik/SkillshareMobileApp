# SkillshareMobileApp
SkillShare is a smart service-booking platform that connects users with skilled workers based on their expertise, availability, and location. It features feedback-based ratings, AI-generated performance summaries, and an easy booking system to help users choose the best workers quickly and confidently.
A mobile application that allows users to share and learn skills.
## Features
- User registration and login
- Post and browse skills
- Simple and user-friendly UI
## Tech Stack
- Flutter
- Dart
- Firebase
## How to Run
1. Clone the repository
2. Open in Android Studio / VS Code
3. Run using emulator or real device
